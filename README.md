# Statmeat
Meat in the Beat

Make Statbeat great again!

# Tools and utilities
Install NodeJS.

Install Firebase tools, https://github.com/firebase/firebase-tools.

`npm install -g firebase-tools`

# To crawl
`node crawl.js`

Crawls Statbeat and generates statmeat.nedb and corresponding data.json.

# To deploy

```
cd Statmeat
firebase deploy
```
